# dijavostore
Dijavo Store is a simple, modern, and responsive e-commerce web application built with **HTML**, **CSS**, and **JavaScript**. The project demonstrates product listing, product details, search functionality, cart management, and contact & about pages with a dark-themed UI.

---

## Project Overview

Dijavo Store is designed to showcase a small e-commerce platform where users can:

- Browse products on the home page.
- Search for products using the navbar search.
- View detailed information for each product.
- Add products to the cart and manage quantities.
- Contact the store through a contact form.
- Learn about the store through the About page.

The project uses **localStorage** to store cart data, providing a simple client-side shopping cart experience.

---

## Features

- **Responsive Navigation Bar**  
  Includes brand logo, links to pages, search bar, and cart icon.

- **Product Listing (Home Page)**  
  - Displays products in a grid layout.
  - Click on product to view details.
  - Add-to-cart button available.

- **Product Details Page**  
  - Shows product image, description, price.
  - Add to cart button.

- **Search Functionality**  
  - Live search in the navbar.
  - Click on search results to navigate to the product page.

- **Shopping Cart**  
  - View all added products.
  - Update quantities.
  - See total price.
  - Stored in browser's localStorage.

- **Contact Page**  
  - Contact form with name, email, subject, and message.
  - Displays contact information.

- **About Page**  
  - Overview of the store.
  - Image section.

- **Dark Theme UI**  
  - Black background.
  - White text.
  - Red hover and action highlights.

---

## Technologies Used

- **HTML5** - Structure of pages.  
- **CSS3** - Styling, grid layouts, dark theme.  
- **JavaScript** - DOM manipulation, localStorage, event handling.
